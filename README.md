## 😎 진행 방법

1. [저장소](#✔-스터디-개요)를 `fork` 합니다.
2. 챕터별로 `개인적으로 학습`한 뒤, 학습한 내용을 `자유 형식(.pdf, .md 등)`으로 `챕터/섹션별 폴더`에 맞게 업로드합니다.
   - 파일명은 아래 예시와 같이 작성합니다.
   - 만약, 2챕터 1섹션을 학습했다?
     - 2-1\_전인혁.md
3. 원본 저장소로 `Pull Request`를 한다. (매주 월요일 오전 09:00까지 !)
